# Trader Behavior Insights

This project analyzes the relationship between Bitcoin Market Sentiment (Fear/Greed Index) and Historical Trader Data to uncover patterns and drive smarter trading strategies.

## Setup

1.  **Data**: Ensure `historical_data.csv` and `fear_greed_index.csv` are in the `data/` directory.
2.  **Environment**: Install dependencies (e.g., `pandas`, `matplotlib`, `seaborn`).

## Running the Analysis

Run the analysis script:
```bash
python src/analysis.py
```

This will generate a report in `report.md`.
